Quick Setup Guide:

  1. Enable the Gmail API in the Google Cloud Console.
  2. Configure your OAuth consent screen (internal/external) and add scopes: gmail.readonly, gmail.compose, gmail.send, gmail.modify.
  3. Create OAuth 2.0 Client ID credentials for a Web Application.
  4. Set authorized redirect URI to http://localhost:3000/api/auth/callback.
